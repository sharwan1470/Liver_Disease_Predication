About the Project

System Overview

This predictive health system aims to assist in the early identification of liver disease. By assessing blood tests and basic patient metrics (e.g. Bilirubin, Enzymes, Proteins), the system provides an evaluation based on historical clinical dataset outcomes.

📊 Machine Learning Model

The prediction core utilizes a **K-Nearest Neighbors (KNN)** Classifier. KNN functions by analyzing the multi-dimensional clinical parameters of the patient and finding the closest matching profiles in the trained clinical dataset to make a prediction.

Algorithm: KNN Classifier

💡 Dataset Details

The model was trained on the **Indian Liver Patient Dataset (ILPD)**, containing patient records collected from North East India. The records comprise clinical attributes detailing liver enzymes, protein configurations, and bilirubin counts.

Trained Records: 1000+ Profiles
